# 直播间演示视频

将预录 MP4 放置于此目录，命名为 `live-room.mp4`。

- 推荐：30–60 秒珠宝/商品讲解竖屏或横屏视频
- 可用 AI 视频工具生成，或使用任意免版权商品展示素材
- 若文件不存在，H5 直播间将自动显示封面占位图（Unsplash）

生成建议提示词（供 AI 视频工具）：

> A professional live-stream host presenting a jade bracelet under soft studio lighting, e-commerce live room style, 16:9, no text overlay
