# 6.8 演示录屏检查清单

> **量化验收**：彩排时请同步填写 [彩排验收表.md](./彩排验收表.md)（20 项必测、DRS 评分、双买家对抗脚本）。

## 录制前

- [ ] `docker compose up -d mysql redis` 已启动
- [ ] `npm run storage:reset` 或 `cd apps/api && npm run db:seed` 已执行
- [ ] API `:3000`、用户端 `:5173`、管理端 `:5174` 正常运行
- [ ] Chrome 设备模拟器设为 iPhone 14（H5）
- [ ] OBS 配置：管理台窗口 + H5 模拟器（或分镜切换）

## 分镜脚本（约 5 分钟）

| 时间 | 画面 | 操作 |
|------|------|------|
| 0:00 | 管理端登录 | host@example.com 登录 http://localhost:5174 |
| 0:30 | 直播控制台 | 选择演示房间，查看待上架 / 直播中 Tab |
| 1:00 | 添加商品 | 「+ 添加商品」：填写标题、起拍价、加价、封顶、延时 |
| 1:30 | 开播 | 房间「开播」或切品讲解 |
| 2:00 | H5 大厅 | buyer 登录 → 大厅进入直播间 `/m/room/:id` |
| 2:30 | 出价 | 橱窗内连续出价，展示排名与倒计时 |
| 3:00 | 延时 | 倒计时最后 30s 内出价，展示延时提示 |
| 3:30 | 第二买家 | buyer2 超越，「我参与的」展示「被超越」 |
| 4:00 | 成交 | 等待结束或达到封顶 |
| 4:30 | 订单 | 管理端订单管理 + H5「订单」Tab |
| 5:00 | 模拟支付 | H5 点击「模拟支付」完成 |
