# 6.8 演示录屏检查清单

## 录制前

- [ ] `docker compose up -d mysql redis` 已启动
- [ ] `npx prisma migrate deploy` 与 `npm run db:seed` 已执行
- [ ] API `:3000`、Web `:5173` 正常运行
- [ ] Chrome 设备模拟器设为 iPhone 14（H5）
- [ ] OBS 配置：管理台窗口 + H5 模拟器（或分镜切换）

## 分镜脚本（约 5 分钟）

| 时间 | 画面 | 操作 |
|------|------|------|
| 0:00 | 管理台登录 | host@example.com 登录 |
| 0:30 | 商品管理 | 展示已有商品或新建「翡翠手镯」 |
| 1:00 | 创建场次 | 起拍价 100、加价 10、封顶 5000、延时 15s |
| 1:30 | 开播 | 点击「开播」 |
| 2:00 | H5 直播间 | buyer 登录，进入 `/m/live/:id` |
| 2:30 | 出价 | 连续出价，展示排名与倒计时 |
| 3:00 | 延时 | 倒计时最后 30s 内出价，展示延时提示 |
| 3:30 | 第二买家 | 另一账号超越，展示「被超越」提醒 |
| 4:00 | 成交 | 等待结束或达到封顶 |
| 4:30 | 订单 | 管理台订单列表 + H5「我的订单」 |
| 5:00 | 模拟支付 | H5 点击「模拟支付」完成 |
