## 1. 演示就绪分（DRS）

```
DRS = (必测步骤通过数 / 20) × 100%
```

| DRS | 结论 |
|-----|------|
| ≥ 95%（19/20 及以上） | **可录屏**（允许 1 项非关键偶发失败） |
| 80%–94% | 修复后再彩排 |
| < 80% | 存在阻塞问题，不可演示 |

## 6. 缺陷分级

| 级别 | 定义 | 处理 |
|------|------|------|
| **P0** | 无法完成出价 / 成交 / 订单任一步 | 对应必测步 Fail，禁止录屏 |
| **P1** | 功能有但超阈值（如 WS > 5s） | 必测步 Fail，修后再彩排 |
| **P2** | 文案、样式、非主路径 | 不计入 20 步，单独列表 |
